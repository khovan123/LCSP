# Test Scenario Catalogue

## Purpose

This catalogue defines stable, behavior-focused test scenarios for LCSP Phase 3.1. It is a design catalogue only and does not create test source.

## Scenario Fields

Each scenario includes stable ID, title, priority, scope, preconditions, test inputs, expected workflow state, expected evidence/audit, blocking condition, linked UC/FR/BR/NFR and validation dependency.
